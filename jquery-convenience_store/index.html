Convenience Store -

A jQuery plugin provides Japanese convenience store styles.

Usage

<script type="text/javascript" src="js/jquery.js"></script>
<script type="text/javascript" src="js/jquery.convenience_store.js"></script>
<script type="text/javascript">
$().ready(function() {
	$(selector).convenience_store('');
});
</script>

Brands

This plugin needs one argment to select convenience-store-brand.

available brands seveneleven
familymart
lawson
owson
